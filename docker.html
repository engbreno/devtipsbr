Docker - Comandos Essenciais 🐳

Gerenciamento de Containers 📦

Executar um container

docker run [opções] imagem
  • -d: Executar em background
  • -p 8080:80: Mapear porta 8080 do host para 80 do container
  • --name meu-container: Definir nome do container

Listar containers

docker ps # containers ativos
docker ps -a # todos os containers

Gerenciar containers

docker start container_id
docker stop container_id
docker restart container_id
docker rm container_id # remover container

Imagens 🖼️

Gerenciar imagens

docker images # listar imagens
docker pull imagem # baixar imagem
docker rmi imagem # remover imagem

Construir imagem

docker build -t nome-imagem .

Docker Compose 🔧

Comandos básicos

docker-compose up -d # iniciar serviços
docker-compose down # parar e remover
docker-compose logs # ver logs

Dicas Úteis 💡

  1. Use docker system prune para limpar recursos não utilizados
  2. Utilize docker-compose para múltiplos containers
  3. Sempre defina tags específicas para imagens
  4. Use .dockerignore para excluir arquivos desnecessários
