.NET - Guia Rápido 🎯

Criar Projetos 🆕

Tipos de Projetos

dotnet new console # Aplicação console
dotnet new webapi # API Web
dotnet new mvc # Aplicação MVC
dotnet new classlib # Biblioteca de classes

Opções comuns

dotnet new console -n MeuProjeto # Define nome
dotnet new webapi -o MinhaPasta # Define pasta

Gerenciar Projetos 🛠️

Comandos básicos

dotnet build # Compilar projeto
dotnet run # Executar projeto
dotnet test # Executar testes
dotnet watch run # Executar com hot-reload

Gerenciar pacotes

dotnet add package NomePacote
dotnet remove package NomePacote
dotnet restore # Restaurar dependências

Soluções (.sln) 📁

Gerenciar soluções

dotnet new sln # Criar solução
dotnet sln add Projeto.csproj # Adicionar projeto
dotnet sln remove Projeto.csproj # Remover projeto

Entity Framework Core 🗃️

Comandos principais

dotnet ef migrations add NomeMigracao
dotnet ef database update
dotnet ef migrations remove

Dicas Úteis 💡

  1. Use dotnet new gitignore para criar .gitignore
  2. Configure launchSettings.json para perfis diferentes
  3. Use dotnet tool install para ferramentas globais
  4. Mantenha o SDK atualizado
